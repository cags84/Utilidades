h1. Para la instalaciÃ³n se debe hacer lo siguiente:

h2. Crear un directorio en el root del servidor
	
	@#mkdir /root/scripts@

h2. Ingresar al directorio

	@#cd /root/scripts@
 
h2. Descargar los archivos
	
	@# wget https://github.com/cags84/Utilidades/archive/master.zip@

h2. Descomprimimos ambos archivos

	@#unzip master.zip@


h2. Le damos permisos a los archivos

	@#chmod +x *.sh@

h2. Editamos los archivos y colocamos nuestras propias reglas

	@#vim /root/scripts/start.sh@
	@#vim /root/scripts/stop.sh@

h2. Cargamos al rc.local, para que arranque con la maquina:

	@#echo '/root/scripts/start.sh' >> /etc/rc.local@

h2. Como lo arrancamos desde la terminal?

	@#sh /root/scripts/start.sh@

h2. Como limpiamos las reglas y para el firewall?

	@#sh /root/scripts/stop.fw@
	
h2. Para ver las reglas

	@#iptables -L -nv@
